# Brand Identity

## 1. Company Name
**SkyForge Cloud**

A modern, reliable, automation‑focused cloud platform built on Proxmox.

---

## 2. Logo Concept
**Simple Concept:**  
A minimal icon showing a **cloud with a gear inside it**, representing automation + cloud services.

**Description:**  
- Cloud outline = your virtual infrastructure  
- Gear = automation, API, provisioning  
- Clean, flat style so it works in documentation and diagrams  

(You don’t need to draw it — just describing the concept is enough for Deliverable 1.)

---

## 3. Tagline
**“Automate. Deploy. Scale.”**

Short, professional, and directly tied to your project’s purpose.

---

## 4. Brand Summary
SkyForge Cloud provides fast, automated virtual machine provisioning using Proxmox, templates, and API‑driven workflows. The brand focuses on simplicity, speed, and reliability for small teams and student developers.
